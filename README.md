# 👨‍💻 Інтерактивний Телеграм-бот Резюме

Цей проект — персональний бот-резюме, створений на Python за допомогою бібліотеки `aiogram 3.x`. Він дозволяє потенційним замовникам та рекрутерам швидко ознайомитися з вашими навичками, отримати копію резюме (PDF) та зв'язатися з вами одним натисканням кнопки.

У проект вбудовано простий веб-сервер на `aiohttp` для проходження "health checks" на хостингах на кшталт Render, що дозволяє боту працювати стабільно. Також бот підтримує логування дій користувачів у базу даних.

### 🔥 Функціонал

* **👤 Резюме:** Надсилає файл резюме у форматі PDF (`Ризюме.pdf`) безпосередньо в чат.
* **👨‍💻 Посилання на GitHub:** Надає активне посилання на профіль розробника.
* **📞 Зв'язок зі мною:** Показує контактну інформацію (номер телефону, e-mail, логін в Telegram).
* **📊 Логування:** Зберігає ID, ім'я та дії користувачів у базі даних (використовується модуль `database`).

---

### 💻 Технологічний Стек

* **Мова:** Python 3.9+
* **Фреймворк бота:** [aiogram 3.x](https://docs.aiogram.dev/en/latest/) (асинхронна бібліотека)
* **Веб-сервер:** [aiohttp](https://docs.aiohttp.org/en/stable/)
* **Керування конфігурацією:** `python-dotenv` (для роботи з токеном бота)
* **postgreSQL:**  Для збереження інформації про користувачів

### 🚀 Запуск та Налаштування

Для запуску бота на своєму локальному комп'ютері або сервері, виконайте наступні кроки.

#### 1. Клонування репозиторію

Спочатку завантажте цей проект до себе:

```bash
git clone [https://github.com/Nikita-tg-python/my_tg_bots.git](https://github.com/Nikita-tg-python/my_tg_bots.git)
cd my_tg_bots
```
#### 2. Встановлення залежностей

Створіть віртуальне оточення та встановіть бібліотеки, необхідні для роботи бота:

```bash
python3 -m venv venv
# Активація для Linux/Ubuntu:
source venv/bin/activate
pip install aiogram aiohttp python-dotenv
```
#### 3. Налаштування файлу конфігурації
Створіть файл .env в кореневій папці проекту (там, де знаходиться головний .py файл бота).
Git не буде відстежувати цей файл через .gitignore, тому ваш токен буде в безпеці.
Додайте до нього наступний рядок:
BOT_TOKEN=ВАШ_ТОКЕН_ВІД_BOTFATHER

#### 4. Підготовка файлів
Переконайтеся, що в кореневій папці проекту знаходиться файл вашого резюме під назвою Ризюме.pdf

#### 5. Запуск бота
```bash
python tg_bot.py
```
